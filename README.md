# 📖 My Dictionary

A modern dictionary web application built using **HTML, CSS, and JavaScript** that allows users to search for word meanings, phonetics, examples, and other linguistic information using the Free Dictionary API.

## 🚀 Features

* Search meanings of English words
* View phonetic pronunciation
* Display part of speech
* Example sentence support
* Error handling for invalid words
* Responsive design
* Fast API-based search
* Clean and intuitive user interface

## ⚙️ How It Works

1. User enters a word.
2. The application sends a request to the Dictionary API.
3. The API returns information about the word.
4. The application displays:

   * Meaning
   * Phonetic pronunciation
   * Part of speech
   * Example sentence

## 🔗 API Used

Free Dictionary API

https://dictionaryapi.dev/

Example Request:

```text
https://api.dictionaryapi.dev/api/v2/entries/en/example
```

## 🏃 Running Locally

Clone the repository:

```bash
git clone https://github.com/anuragrajpoott/dictionary.git
```

Navigate to the project folder:

```bash
cd dictionary-app
```

Open `index.html` in your browser.

No installation or build process required.
